# RssReaderMac

A clean, focused RSS reader for macOS.

## Features

- **Clean Reading** - Distraction-free article view with reader mode
- **Smart Folders** - Auto-organize articles with custom filter rules
- **iCloud Sync** - Sync feeds and read state across your Macs
- **Native Experience** - Built with SwiftUI, follows Apple Human Interface Guidelines
- **OPML Support** - Import and export your feed subscriptions

## Requirements

- macOS 13.0 or later

## Building

Open `RssReaderMac.xcodeproj` in Xcode 15+ and build.
